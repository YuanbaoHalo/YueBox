# Project Instructions

## Goal
Build a simple playable puzzle game inspired by Tetris.
Yue Opera theme will be added later.

## Current Stage
Only build MVP first.
Focus on making the game playable.

## Development Rules
- Implement step by step
- Do not build too many features at once
- Always explain before writing code
- After coding, explain how to test
- Do not modify unrelated files

## Code Rules
- Use TypeScript
- Keep logic simple
- Separate game logic and UI

## Product Style
- Keep the game simple and beginner-friendly
- Avoid over-engineering
